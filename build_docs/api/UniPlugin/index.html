UniPlugin

Wraps a Roblox Plugin instance to provide a shared export/import system between plugins.

Functions​

exportAs​

</>
UniPlugin.exportAs(
self: UniPlugin,
alias: string--

The name to register this plugin under

) → UniPluginConnection--

A connection that can be closed to unregister the export

Registers this UniPlugin under alias in the shared exports table, making it discoverable by other plugins calling withImports. Fires the global ExportOpened event so any waiting withImports listeners are unblocked.

The returned listener's close removes the export and can be called manually; it is also called automatically when the plugin unloads or is destroyed.

Example Usage:

local UniPlugin = require(path.to.module)(plugin)

UniPlugin:setApi("output", {
	print = print,
	warn = warn,
	error = error
})
UniPlugin:setApi("requireInMyContext", {
	execute = function(module) return require(module) end
})
UniPlugin:exportAs("athar-adv/MyPlugin")

Errors

TypeDescription
"UniPlugin '{self.plugin.Name}' is closed, cannot call UniPlugin:exportAs(...)"Occurs if UniPlugin.isClosed is true.

withImports​

</>
UniPlugin.withImports(
self: UniPlugin,
aliases: {string},--

List of plugin export aliases to wait for

handler: (
plugins: () → ...UniPlugin,
bindOnImportClosed: (fn: () → ()) → ()
) → ()--

Called once all aliases are available; re-runs if the group is torn down and re-established

) → UniPluginConnection--

A connection that can be closed to stop tracking imports

Waits for all plugins in aliases to be exported, then calls handler with two arguments:

  • plugins() — returns all resolved UniPlugins in the same order as aliases via multiple returns.
  • bindOnImportClosed(fn) — registers a single callback (overwriting any previous one) that fires whenever this import group is torn down, whether because one of the imported plugins closed naturally or because the returned listener was closed manually.

The handler re-runs automatically each time all aliases become available again after a close. Duplicate aliases are silently deduplicated. Passing an empty aliases table is a no-op and returns an inert connection.

Close the returned connection to stop listening entirely and tear down the current import group.

Example Usage:

local UniPlugin = require(path.to.module)(plugin)

UniPlugin:withImports({"athar-adv/Jexplorer", "athar-adv/SimpleComplete"}, function(plugins, onImportClosed)
	local jexplorer, simpleComplete = plugins()

	local items = {}
	do
		local manager = jexplorer:getApi("manager")
		manager.setSetting("VirtualScrollFrameBuffer", 20)
		
		table.insert(items, manager.ExplorerFrame.ClassIconStyleRuleCreated:Connect(function(rule)
			rule:SetProperty("Image", "rbxassetid://1234")
		end))
	end
	do
		local selectionImports = simpleComplete:getApi("selectionImports")

		selectionImports.triggerAtCurrentPosition()
	end
	onImportClosed(function()
		for _, v in items do
			v:Disconnect()
		end
	end)
end)

Errors

TypeDescription
"UniPlugin '{self.plugin.Name}' is closed, cannot call UniPlugin:withImports(...)"Occurs if UniPlugin.isClosed is true.

onClose​

</>
UniPlugin.onClose(
self: UniPlugin,
fn: () → ()--

Callback to invoke on plugin teardown

) → UniPluginConnection--

A connection that can be closed to cancel the subscription

Registers fn to be called when this plugin is torn down (i.e. when the plugin unloads or is destroyed). Multiple listeners can be registered and all will fire on close.

The returned listener's close unregisters fn and can be used to cancel the subscription before the plugin closes.

Example Usage:

local UniPlugin = require(path.to.module)(plugin)

UniPlugin:onClose(function()
	print("Whyd you close my plugin :(")
end)

getApi​

</>
UniPlugin.getApi(
self: UniPlugin,
alias: string?--

The alias to look up. Defaults to "main"

) → Api--

The stored API table

Returns the Api stored under alias (defaults to "main"). Errors if no Api has been set for that alias.

Example Usage:

local UniPlugin = require(path.to.module)(plugin)

UniPlugin:withImports({"athar-adv/Jexplorer", "athar-adv/SimpleComplete"}, function(plugins, onImportClosed)
	local jexplorer, simpleComplete = plugins()

	local items = {}
	do
		local manager = jexplorer:getApi("manager")
		print(manager.executeJqlQuery("#?player"))

		local dependencies = jexplorer:getApi("dependencies")
		print(dependencies.GitApi)
	end
	do
		local editor = simpleComplete:getApi("editor")
		print(editor.getOpenDocValue().currentImportedModules)
	end
end)

close​

</>
UniPlugin.close(self: UniPlugin) → ()

Closes this UniPlugin. Cleans up any connections, fires onClose handlers, and makes all methods error on subsequent calls.

Example Usage:

local UniPlugin = require(path.to.module)(plugin)

--[[snip!]]

closePluginButton.Activated:Connect(function()
	UniPlugin:close()
end)

UniPlugin:onClose(function()
	for _, v in connections do v:Disconnect() end
end)

openUniPlugin​

</>
UniPlugin.openUniPlugin(
plugin: Plugin--

The Roblox Plugin instance to wrap

) → UniPlugin

Returns the UniPlugin instance for the given plugin, creating it if one doesn't already exist. Only one UniPlugin is ever created per Plugin instance for the lifetime of the studio session.

Automatically connects to plugin.Unloading and plugin.Destroying to fire all onClose listeners and close all active withImports and exportAs listeners when the plugin is torn down.
